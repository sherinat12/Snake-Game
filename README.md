A simple console-based Snake Game developed using the C programming language.  
The game runs in the terminal and allows the player to control a snake, eat fruits, and increase the score.

---

## Features

- Console-based gameplay
- Snake movement using keyboard
- Score tracking
- Fruit generation
- Game over detection
- Tail growth after eating fruit

---

## Technologies Used

- C Programming
- Functions
- Arrays
- Loops
- Conditional Statements

---

## Controls

| Key | Action |
|------|---------|
| W | Move Up |
| S | Move Down |
| A | Move Left |
| D | Move Right |
| X | Exit Game |

---

## How to Run

### Step 1: Compile the Program

```bash
gcc snakegame.c -o snakegame
```

### Step 2: Run the Program

For Linux / Mac:

```bash
./snakegame
```

For Windows:

```bash
snakegame.exe
```
